# Project Planner

A multi-project task management application built with React and Convex.

## Run locally

- Clone this repository.
- Navigate to the project directory.
- Install dependencies using `pnpm install`.
- Start the Convex dev server with `npx convex dev`.
- Start the development server with `pnpm run dev` (in a separate terminal).
- Open your browser and navigate to `http://localhost:5173`.
